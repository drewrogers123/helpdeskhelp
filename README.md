MSP Toolkit
A collection of scripts, shortcuts, how-to guides, and templates to make day-to-day MSP work faster and more consistent.

What's in Here:
AutoHotKey scripts, scripts to run in the main browser you use, and script that need to be running in a seperate browser.

The scripts were designed, tested, and implimented in ViolentMonkey. I use Firefox as my main browser, and Edge as a second browser.
