MSP Toolkit
A collection of scripts, shortcuts, how-to guides, and templates to make day-to-day MSP work faster and more consistent.

What's in Here
scripts: PowerShell, shell, and cloud automation scripts
shortcuts: Keyboard shortcuts, text expander snippets
howtos: Step-by-step guides for common tasks and procedures
templates: Runbooks, email templates, ticket macros, SOPs
Each folder has its own README with more detail.

Using the Scripts
Most scripts have a config section at the top. Check the comments for any requirements (PowerShell version, modules, permissions, etc.).
**Always test in a non-production environment before running against client systems.**
