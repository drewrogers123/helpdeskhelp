A set of AutoHotkey v2 scripts built to streamline day-to-day helpdesk operations. 
Covers browser navigation shortcuts, dual clipboard management, hyperlink and credential formatting, and scroll acceleration, all aimed at reducing repetitive keystrokes and keeping your hands on the keyboard when working through tickets. 

Requires AutoHotkey v2 to run. To use, launch individual scripts or add them to your Windows startup folder so they load automatically on boot.
